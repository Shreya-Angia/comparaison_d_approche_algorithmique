Fonction qui permet de gérer les actions que peut réaliser un praticien. 

La fonction demande le nom du praticien, le recherche dans toutes les spécialités, puis propose les actions qu\textquotesingle{}il peut réaliser (appeler un patient, afficher les patients en ordre lexicographique, afficher les patients en ordre de leur date de visite, afficher les patients qui ne sont pas venus depuis une date donnée, faire de la place dans la file d\textquotesingle{}attente). La fonction sauvegarde les données dans un fichier quand le praticien quitte.

Fonction qui gère les actions que peuvent réaliser les secrétaires. 

Cette fonction affiche le menu des secrétaires et permet de choisir une action. Les actions possibles sont l\textquotesingle{}ajout d\textquotesingle{}un praticien, la suppression d\textquotesingle{}un praticien, l\textquotesingle{}affichage des spécialités et l\textquotesingle{}affichage des patients en attente. La fonction sauvegarde également les données dans le fichier "{}sources/\+Centre.\+txt"{} quand la secretaire choisit de quitter.
