Ajoute un patient a la fin d'une file.

La fonction ajouterPatientF prend en paramètre une file et un patient. Elle alloue de la mémoire pour un maillon de file, puis elle met a jour les pointeurs de tête et de queue de la file en fonction de si la file est vide ou non.

Paramètres
fLa file dans laquelle on veut ajouter le patient.
pLe patient a ajouter.
Renvoie
La file avec le patient ajouté.
