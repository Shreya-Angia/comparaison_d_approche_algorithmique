Recherche dichotomique d'une specialité dans un tableau de pointeur.

Recherche une spécialité dans un tableau de pointeur vers des spécialités. La recherche est faite en utilisant l'algorithme de recherche dichotomique. Si la spécialité est trouvée, la fonction renvoie l'indice de la spécialité dans le tableau. Sinon, la fonction renvoie -1.

Paramètres
tSpeLe tableau de pointeur vers les spécialités.
nomLe nom de la spécialité à rechercher.
nbSpeLe nombre de spécialités dans le tableau.
Renvoie
L'indice de la spécialité dans le tableau si elle est trouvée, -1 sinon.
