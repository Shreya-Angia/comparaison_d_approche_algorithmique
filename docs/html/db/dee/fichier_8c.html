Chargement d'un fichier de spécialités, de leurs praticiens et de leurs patients.

La fonction charge un fichier de spécialités, de leurs praticiens et de leurs patients. Elle lit le fichier et stock les informations dans des structures de type Specialite, Praticien et Patient.

Paramètres
flotLe fichier à charger.
tabSpeLe tableau des spécialités.
nbSpeLe nombre de spécialités dans le tableau.
maxSpeLa capacité maximale du tableau.
tlogLe nombre de spécialités déjà dans le tableau.
tphyLe nombre de spécialités que l'on peut encore ajouter dans le tableau.
Renvoie
1 si le chargement est réussi, -1 si l'un des arguments est NULL, -2 si la réallocation du tableau échoue.
