◆ enregistrer1Patient()

void enregistrer1Patient ( Specialite **  tspe,
int  nbSpe 
)

Enregistre un patient dans la spécialité demandée.

Paramètres
tspeUn tableau de pointeur vers des spécialités.
nbSpeLe nombre de spécialités dans le tableau.

On demande le nom du patient, la spécialité demandée, et si le patient a un praticien référent. Si c'est le cas, on recherche le praticien référent et on ajoute le patient à sa file d'attente. Sinon, on recherche un praticien disponible dans la spécialité et on ajoute le patient à sa patientèle et à sa file d'attente.

◆ globaleAjouterPraticien()

void globaleAjouterPraticien ( Specialite **  tabSpe,
int *  nbSpe 
)

Fonction qui ajoute un praticien dans la liste des praticiens d'une spécialité.

La fonction demande le nom de la spécialité et le nom du praticien à ajouter. Si la spécialité n'existe pas, elle est créée. Le praticien est ajouté à la liste des praticiens de la spécialité.

Paramètres
tabSpeLe tableau de spécialités.
nbSpeLe nombre de spécialités dans le tableau.

Affiche la liste des patients d'une spécialité.

La fonction demande le nom de la spécialité, recherche le dans le tableau des spécialités et affiche la liste des patients de la spécialité demandée.

Paramètres
tabSpeLe tableau de spécialités.

◆ supprimer1Praticien()

void supprimer1Praticien ( Specialite **  tspe,
int *  nbSpe 
)

Supprime un praticien d'une spécialité.

Demande le nom du praticien et sa spécialité, puis supprime le praticien de la liste des praticiens de cette spécialité. Si la liste des praticiens est vide après la suppression, on supprime la spécialité.

Paramètres
tspeLe tableau des spécialités.
nbSpeLe nombre de spécialités dans le tableau.
