void globalePraticien ( Specialite **  tabSpe,
int  nbSpe 
)

Fonction qui permet de gérer les actions que peut réaliser un praticien.

La fonction demande le nom du praticien, le recherche dans toutes les spécialités, puis propose les actions qu'il peut réaliser (appeler un patient, afficher les patients en ordre lexicographique, afficher les patients en ordre de leur date de visite, afficher les patients qui ne sont pas venus depuis une date donnée, faire de la place dans la file d'attente). La fonction sauvegarde les données dans un fichier quand le praticien quitte.

Paramètres
tabSpeLe tableau de pointeur vers les spécialités.
nbSpeLe nombre de spécialités dans le tableau.

◆ globaleSecretaire()

void globaleSecretaire ( Specialite **  tabSpe,
int *  nbSpe 
)

Fonction qui gère les actions que peuvent réaliser les secrétaires.

Cette fonction affiche le menu des secrétaires et permet de choisir une action. Les actions possibles sont l'ajout d'un praticien, la suppression d'un praticien, l'affichage des spécialités et l'affichage des patients en attente. La fonction sauvegarde également les données dans le fichier "sources/Centre.txt" quand la secretaire choisit de quitter.

Paramètres
tabSpeLe tableau de spécialités
nbSpeLe nombre de spécialités dans le tableau
